# PRD: ElectricElite
## Core Purpose
Automate the gap between inventory entry and marketplace-ready copywriting.

## MVP Scope
1. **Dashboard:** KPI cards (Total Products, Stock Value, Low Stock Alerts) + Activity Timeline.
2. **Inventory CRUD:** Add/Edit/Delete products with image uploads and manual specs.
3. **AI Intake:** Image-to-Data auto-filling using Vision API.
4. **AI Generator:** Transform specs into Amazon/Flipkart listings & social media copy.
5. **Analytics:** Basic category distribution charts.
6. **Settings:** Light/Dark mode toggle.
